## Definitions

### Register
A small, fast storage location inside the CPU used to hold temporary data during execution.

### Memory
A larger storage area that holds instructions and data for the CPU. Slower than registers.

### 8-bit
A unit of data that consists of 8 binary digits (bits), allowing values from 0 to 255.

### LOAD
An instruction that copies data from memory into a register.

### STORE
An instruction that copies data from a register into memory.

### ADD
An instruction that adds two values and stores the result in a register.

### Program Counter (PC)
A register that keeps track of the address of the next instruction to execute.
